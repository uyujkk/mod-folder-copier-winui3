# Mod 文件复制器 v2.2.4 使用说明

## 1. 运行方法

1. 解压发布包到任意文件夹
2. 双击 `ModFolderCopier.exe`
3. 首次运行后，先设置：
   - Mod 存储文件夹
   - 目标文件夹
   - 启动器地址（可选）
4. 点击“刷新目录”

## 2. Mod 文件夹结构要求

程序按两层目录工作，推荐结构如下：

```text
Mod 存储文件夹
├─ 分类A
│  ├─ Mod1
│  ├─ Mod2
├─ 分类B
│  ├─ Mod3
│  └─ Mod4
```

- 第一层：分类目录
- 第二层：实际操作的 Mod 文件夹

## 3. 基本使用流程

1. 选择 Mod 存储文件夹
2. 选择目标文件夹
3. 点击“刷新目录”
4. 在左侧选择第一层分类
5. 在中间选择第二层 Mod
6. 双击第二层 Mod，或点击“复制当前第二层文件夹”

说明：

- 如果目标文件夹中没有同名目录，会执行复制
- 如果目标文件夹中已经有同名目录，会执行移除

## 4. ZIP 导入

1. 先选中一个第二层 Mod
2. 点击“导入 ZIP 到当前第二层”
3. 选择 `.zip` 压缩包

程序会把压缩包内容解压到当前 Mod 文件夹。

## 5. 预览图

程序优先读取以下命名的图片：

- `preview.*`
- `cover.*`
- `thumbnail.*`
- `image.*`

如果没有这些文件，会尝试显示当前 Mod 文件夹中的第一张图片。

也可以直接把图片拖到右侧预览区，程序会自动将其设为当前 Mod 的预览图。

## 6. 快捷键与描述

每个第二层 Mod 都可以单独记录快捷键与描述。

支持：

- 单个按键，例如 `1`、`Q`、`F1`
- 组合键，例如 `Ctrl+1`

最多支持 10 行。

## 7. Mod 链接

预览图下方支持绑定当前 Mod 的专属链接。

点击快速访问按钮后，会调用系统默认浏览器打开该链接。

## 8. 启动器

如果你设置了启动器地址，点击“运行启动器”即可启动对应程序。

## 9. 注意事项

- 本工具为非官方工具
- 与 XXMI、游戏发行商及相关开发者无从属或官方合作关系
- 如果首次运行时遇到 Windows SmartScreen 提示，可以在确认来源可信后选择继续运行
