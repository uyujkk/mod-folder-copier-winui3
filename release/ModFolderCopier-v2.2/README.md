# Mod 文件复制器 WinUI 3 版 v2.2

> Non-official notice:
> This project is an unofficial fan-made tool. It is not affiliated with, endorsed by, or sponsored by XXMI, any game publisher, or any related developers.

这是发布包内附带的使用说明。解压后直接运行 `ModFolderCopier.exe` 即可。

## 快速开始

1. 双击 `ModFolderCopier.exe`
2. 设置主文件夹
3. 设置副文件夹
4. 如有需要，设置启动器地址
5. 点击“刷新目录”
6. 选择第一层分类和第二层 Mod
7. 双击第二层 Mod，或点击“复制当前第二层子文件夹”

## 主要功能

- 两层 Mod 文件夹管理
- 第二层 Mod 复制 / 移除切换
- ZIP 导入
- 预览图显示
- 拖拽图片设置预览图
- 每个 Mod 的快捷键说明
- 外部启动器集成

## 说明

- `WinUI3` 文件夹内是运行所需依赖，请不要删除
- 如果无法运行，请确保所有文件已完整解压
- 如果看不到图片预览，请检查 Mod 文件夹内是否包含常见图片格式文件
