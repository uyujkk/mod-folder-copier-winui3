# Mod 文件复制器 v2.2 使用说明

## 1. 运行方法

1. 解压发布包到任意文件夹
2. 双击 `ModFolderCopier.exe`
3. 首次运行后，先设置：
   - 主文件夹
   - 副文件夹
   - 启动器地址（可选）
4. 点击“刷新目录”

## 2. 主文件夹结构要求

程序按两层目录工作，推荐结构如下：

```text
主文件夹
├─ 分类A
│  ├─ Mod1
│  ├─ Mod2
├─ 分类B
│  ├─ Mod3
│  └─ Mod4
```

- 第一层：分类目录
- 第二层：实际操作的 Mod 文件夹

## 3. 基本使用流程

1. 选择主文件夹
2. 选择副文件夹
3. 点击“刷新目录”
4. 在左侧选择第一层分类
5. 在中间选择第二层 Mod
6. 双击第二层 Mod，或点击“复制当前第二层子文件夹”

## 4. ZIP 导入

1. 先选中一个第二层 Mod
2. 点击“导入 ZIP 到当前第二层”
3. 选择 `.zip` 压缩包

## 5. 预览图

可以直接把图片拖到右侧预览区，程序会自动将其保存为当前 Mod 的预览图。

## 6. 注意事项

- 不要删除 `WinUI3` 文件夹内的运行时文件
- 如果无法运行，请确保所有文件已完整解压
- 本工具为非官方工具，不隶属于 XXMI、游戏发行商或相关开发者
