# Mod 文件复制器 v2.2.3 使用说明

## 1. 运行方法

1. 解压发布包到任意文件夹
2. 双击 `ModFolderCopier.exe`
3. 首次运行后，先设置：
   - Mod 存储文件夹
   - 目标文件夹
   - 启动器地址（可选）
4. 点击“刷新目录”

## 2. 基本使用流程

1. 选择 Mod 存储文件夹
2. 选择目标文件夹
3. 点击“刷新目录”
4. 选择第一层分类和第二层 Mod
5. 双击第二层 Mod，或点击复制按钮

## 3. 其他功能

- 支持导入 ZIP 到当前 Mod
- 支持拖动图片设置预览图
- 支持记录快捷键与描述
- 支持为每个 Mod 绑定链接并快速访问
- 支持运行外部启动器
