# Mod Folder Copier v3.0-beta 更新文案

## 中文

### 版本信息
- 版本：`v3.0-beta`
- 类型：Beta 测试构建
- 说明：这一版主要用于验证新架构，不替代当前正式版。

### 本次更新重点
- 完成一期 Beta 基线，正式引入多仓库工作区架构。
- 新增一级侧边栏与二级侧边栏结构，用于模块切换与仓库切换。
- 仪表板、仓库页、设置页已经完成基础落地。
- 仓库支持新建、编辑、重命名、删除与切换。
- 仓库配置已独立保存，可记录名称、Mod 仓库路径、目标路径、启动器路径等信息。
- 设置页已承接主题切换、语言切换、GitHub 仓库入口与 GitHub 更新检查。
- 已开始二期准备，`在线 Mod` 页面骨架已接入 Beta 版本。

### 当前 Beta 状态
- 可作为新架构的验证版本继续推进。
- 已可正常编译，并可打包为独立压缩包进行测试。
- 目前重点仍是架构验证、交互打磨和在线能力预研。

### 已知说明
- 这仍然是 Beta 版本，界面细节和部分交互还会继续调整。
- 当前仍存在部分 WinUI 平台告警，但不影响现阶段构建与基本使用。
- 在线 Mod、安装流程与更新追踪尚未完整实现。

## English

### Release Info
- Version: `v3.0-beta`
- Type: Beta test build
- Notes: This build is for validating the new architecture and does not replace the current stable release.

### Highlights
- Phase 1 beta baseline is complete, with the new multi-repository workspace architecture now in place.
- Added a primary sidebar and secondary sidebar for module navigation and repository switching.
- Dashboard, repository view, and settings view are now available in the beta shell.
- Repositories now support create, edit, rename, delete, and switch actions.
- Repository settings are now stored independently, including repository name, mod source path, target path, and launcher path.
- The settings page now hosts theme switching, language switching, GitHub repository access, and GitHub update checking.
- Phase 2 preparation has started, and the `Online Mod` page skeleton is now available in the beta build.

### Current Beta Status
- This build can now serve as the working baseline for the new architecture.
- It compiles successfully and can be packaged into a standalone zip for testing.
- The current focus remains architecture validation, interaction polish, and online feature groundwork.

### Known Notes
- This is still a beta build, and UI details plus some interactions will continue to change.
- Some WinUI platform warnings still exist, but they do not block building or basic usage.
- Online mod browsing, install flow, and update tracking are not fully implemented yet.
